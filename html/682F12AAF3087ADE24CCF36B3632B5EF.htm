Creating a graph

Previous section: Concepts

To plot a line, a scatter or a line-scatter plot, select a V column in the worksheet and press one of the plot buttons in the toolbar or from the menu select Plot→Line (or Plot→Scatter or Plot→Line_Scatter respectively). The data of the selected V column is then plotted versus the corresponding X column. Remember that the corresponding X column is a column with the same group number as the V column, but which is designated as X column.

To create a graph with multiple plot curves, just select multiple V columns in the worksheet before creating the graph. It is even possible to use V columns with different group numbers.
