Installation of Altaxo

Previous section: Altaxo user manual

Installation of Altaxo can be a matter of Copy-and-Paste. If you want to use some advanced possibilities, for instance embedding Altaxo graphs into Word-Documents, you will need to execute some additional steps.

Altaxo requires the .Net framework version 4.8 being installed on your computer. If the AutoUpdate feature is switched on, this should already be the case. Since it is not possible to install .Net framework 4.8 on Windows XP, it is no longer possible to install a current version of Altaxo on Windows XP.
