ISREA This class detrends a spectrum. This is done by fitting a smoothing spline through the spectrum. Then the spectral data that lies above the spline are reduced in amplitude toward the spline. The process is repeated until the resulting curve changes no more.
ISREASerializationSurrogate0 2024-04-16 V0 initial version.
PolynomialDetrending This class detrends all spectra. This is done by fitting a polynomial to the spectrum (x value is simply the index of data point), and then subtracting the fit curve from the spectrum. The degree of the polynomial can be choosen between 0 (the mean is subtracted), 1 (a fitted straight line is subtracted).
