Installation by using a Microsoft installer file (.msi) on Windows 7, 8.1 and 10

Previous section: Installation by Copy-and-Paste on Windows 7, 8.1 and 10

Choosing this installation method requires administrative rights.

  • Make sure you have the latest version of the .Net-Framework installed on your computer. If you have turned on Windows-Updates on your computer, you should already have the newest .Net-Framework version.

  • Download the latest Altaxo .msi file. You can either use the latest stable distribution from this folder, or the latest unstable distribution from this folder.

  • Select the file named Altaxo-x.x.xxxx.x.msi (x is the version number), and download it.

  • Double-click on the downloaded file to install it.

  • Choose installation location and some options.

  • You are done!
