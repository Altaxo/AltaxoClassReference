Introduction

Previous section: Text documents - Markdown in Altaxo

You can create text documents in Altaxo in any folder you want, using Markdown syntax with some extensions.

The really cool feature is that you can include all the 2D and 3D graphs from your Altaxo project, using either absolute path names, or relative path names (relative to the current location of your text document).

The following graph was created using this markdown:

C#
Copy
![](graph://Foo\GRAPH0)

Please note that the Url starts with graph://, which means that the name part which follows is considered as the absolute path to the graph in your Altaxo document. To use relative path names, the Url has to start with graph: (i.e. without the two slashes). There are more rules, e.g. if the path contains spaces and special chars, which will be considered later.
