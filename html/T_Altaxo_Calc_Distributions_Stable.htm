Stable Class

Continuous Univariate Stable distribution. A random variable is said to be stable (or to have a stable distribution) if it has the property that a linear combination of two independent copies of the variable has the same distribution, up to location and scale parameters. For details about this distribution, see Wikipedia - Stable distribution.
