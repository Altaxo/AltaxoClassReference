TryGetCompoundTransformation
Try to get a compound transformation. Use this function when in doubt how many transformations the enumeration yields. The behavior is as follows: if the enumeration is null or empty, the return value is null. If the enumeration contains only one element, the return value is that element. If the enumeration contains multiple elements, the return value is a compound transformation with all elements.
