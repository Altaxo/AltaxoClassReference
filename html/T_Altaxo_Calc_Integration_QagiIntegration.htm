Remarks
This class computes the integral of the function f over the infinite interval (-Infinity,+Infinity). The integral is mapped onto the semi-open interval (0, 1] using the transformation x = (1 - t)/t. It is then integrated using the QAGS algorithm. The normal 21-point Gauss-Kronrod rule of QAGS is replaced by a 15-point rule, because the transformation can generate an integrable singularity at the origin. In this case a lower-order rule is more efficient.

Ref.: Gnu Scientific library reference manual (http://www.gnu.org/software/gsl/)
