VectorMathToROVectorStructAmendedUnshifted Method (Int16, Int16, Int16)

Wraps a Int16[] array to get a struct with an IROVectorT implementation. The wrapping is done lazily, i.e. you can access elements with indices below zero and above Length, which is normally forbidden. The values for that are given as parameters.
