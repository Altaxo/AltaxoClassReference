NLFitLMFunction Delegate

User-supplied subroutine which calculates the functions to minimize. Calculates numberOfYs functions dependent on numberOfParameter parameters and returns the calculated y values in array ys. The value of info should not be changed unless the user wants to terminate execution of LevenbergMarquardtFit. In this case set iflag to a negative integer.
