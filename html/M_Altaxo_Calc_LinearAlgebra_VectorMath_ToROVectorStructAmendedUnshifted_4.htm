VectorMathToROVectorStructAmendedUnshifted(Int32, Int32, Int32) Method

Wraps a Int32[] array to get a struct with an IReadOnlyListT implementation. The wrapping is done lazily, i.e. you can access elements with indices below zero and above Length, which is normally forbidden. The values for that are given as parameters.
