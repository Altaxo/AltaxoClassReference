ItemLocationDirectParentAnchorZ Property

Gets or sets the parent anchor's z position. This is an absolute value (in points = 1/72 inch) or a value relative to the parent's height. The parent anchor point is the point inside the parent item, from which the location of the item is measured. Stricly speaking, the position of the item (as shown in the dialog) is the vector from the parent's anchor point to the local anchor point). A relative value of 0 designates the upper boundary of the parent, a relative value of 0.5 designates the vertical center of the parent, and a relative value of 1 designates the lower boundary of the parent.
