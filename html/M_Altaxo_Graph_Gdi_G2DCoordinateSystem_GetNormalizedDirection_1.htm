Parameters

r0
Type: Altaxo.GraphLogical3D
Logical coordinates of starting point of the isoline.
r1
Type: Altaxo.GraphLogical3D
Logical coordinates of end point of the isoline.
t
Type: SystemDouble
Parameter between 0 and 1 that determines the point on the isoline. A value of 0 denotes the starting point of the isoline, a value of 1 the end point. The logical coordinates are linear interpolated between starting point and end point.
angle
Type: SystemDouble
Angle between direction of the isoline and returned normalized direction vector.
normalizeddirection
Type: Altaxo.GeometryPointD2D
Returns the normalized direction vector,i.e. a vector of norm 1, that has the angle angle to the tangent of the isoline.

Return Value

Type: PointD2D
The location (in layer coordinates) of the isoline point.
