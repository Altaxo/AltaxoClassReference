Installation by Copy-and-Paste on Windows 7, 8.1 and 10

Previous section: Installation of Altaxo

For installation by Copy-and-Paste you don't need administrative rights.

  • Make sure you have the latest version of the .Net-Framework installed on your computer. If you have turned on Windows-Updates on your computer, you should already have the newest .Net-Framework version.

  • Download the latest Altaxo binary distribution. You can either use the latest stable distribution from this folder, or the latest unstable distribution from this folder.

  • Select the file named AltaxoBinaries-x.x.xxxx.x.zip (x is the version number), and download it.

  • Important: before unzipping the downloaded file, make sure that you right-click on the zip-file in Explorer, choose Properties, and then click Unblock.

  • Now you can unzip the content of this file into an empty folder of your choice. This can even be a folder on a USB memory stick.

  • Start Altaxo by double-clicking on AltaxoStartup.exe located in the 'bin' subfolder of the installation directory.

  • To use advanced features, like embedding Altaxo graphs in Word documents, choose from the main menu Tools → Registry → Register Altaxo for OLE/COM functionality. If you have administrative rights, Altaxo is registered for all users. If you do not have administrative rights, Altaxo is registered only for your account.

  • You are done!
