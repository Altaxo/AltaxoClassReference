SuspendableDocumentNodeWithSingleAccumulatedDataT Class

Base class for a suspendable document node. This class stores a single object to accumulate event data. This class supports document nodes that have children, and implements most of the code neccessary to handle child events and to suspend the childs when the parent is suspended.
