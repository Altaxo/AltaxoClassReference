FindCorrespondingCSLineIDWhenChangingCoordinateSystem
When changing the properties of the coordinate systen (e.g. reversing x-axis), the axis including ticks and labels will move from one end of the graph to the other. In order to keep the axes at their location, new CSLineIDs have to be found for the new coordinate system, that correspond to the same axis location in the old coordinate system. This function returns a new CSLineID, or null if no corresponding CSLineID could be found.

Gets the axis line vector. This is a vector pointing from the origin to the axis line, when the layer is assumed to be a square of 2x2x2 size, centered at the origin.
