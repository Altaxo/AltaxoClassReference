IPhysicalBoundaries Methods

The IPhysicalBoundaries type exposes the following members.

Methods
 NameDescription
Add(AltaxoVariant) Processes a single value. If the data value is inside the considered value range, the boundaries are updated and the number of items is increased by one. The function has to return true in this case. On the other hand, if the value is outside the range, the function has to return false.
Add(IPhysicalBoundaries) merged boundaries of another object into this object
Add(IReadableColumn, Int32) Processes a single value from a numeric column col[idx]. If the data value is inside the considered definition range of the values, the boundaries are updated and the number of items is increased by one. The function has to return true in this case. On the other hand, if the value is outside the range, the function has to return false.
