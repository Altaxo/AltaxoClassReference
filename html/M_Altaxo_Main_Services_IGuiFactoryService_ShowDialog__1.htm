Remarks
The presumtions to get this function working are: A controller which implements IMVCAController has to exist.A UserControllerForObjectAttribute has to be assigned to that controller, and the argument has to be the type of the object you want to configure.A GUI control (Windows Forms: UserControl) must exist, to which an UserControlForControllerAttribute is assigned to, and the argument of that attribute has to be the type of the controller.
