Gets or sets the local anchor's x position. This is an absolute value (in points = 1/72 inch) or a value relative to the own item width. A relative value of 0 designates the left boundary of the item, a relative value of 0.5 designates the horizontal center of the item, and a relative value of 1 designates the right boundary of the item.
