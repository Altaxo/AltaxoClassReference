MVCANControllerEditImmutableDocBaseTModel, TView Class

Base class of controllers that edit an immutable document or a document that is a struct. This means that the document can be copied from one member to another without cloning it.

UseDocumentCopy
Sets whether or not a copy of the document is used. If set to true, a copy of the document is used, so if the controller is not applied, all changes can be reverted. If set to false, no copy must be made. The document is directly changed by the controller, and changes can not be reverted. Use the last option if a controller up in the hierarchie has already made a copy of the document.
