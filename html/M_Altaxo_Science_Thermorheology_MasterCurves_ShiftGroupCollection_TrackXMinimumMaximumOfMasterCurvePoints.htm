Tracks the x minimum and x maximum of the master curve points (independently of the interpolation points, for each iteration only those curves which were already considered).

Parameters

idxCurve
Type: SystemInt32
The index of the curve to consider.
shift
Type: SystemDouble
The shift value for this curve.
startNewTracking
Type: SystemBoolean
If set to true, a new tracking will be started, i.e. the xmin and xmax of the curve (under consideration of the shift value) is set as the new tracked xminimum and xmaximum. If false, the xmin and xmax of the curve (under consideration) of the shift value is calculated, and then merged into the tracked xminimum and xmaximum.
