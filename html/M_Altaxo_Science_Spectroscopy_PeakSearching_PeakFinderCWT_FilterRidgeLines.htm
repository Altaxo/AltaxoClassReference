PeakFinderCWTFilterRidgeLines Method

Filter ridge lines according to prescribed criteria. Intended to be used for finding relative maxima.

Namespace: Altaxo.Science.Spectroscopy.PeakSearching
Assembly: AltaxoCore (in AltaxoCore.dll) Version: 4.8.3448.0 (4.8.3448.0)

public static List<RidgeLine> FilterRidgeLines(
	IROMatrix<double> cwt,
	List<RidgeLine> ridge_lines,
	int? window_sizen = null,
	int? min_lengthn = null,
	double min_snr = 1,
	double noise_perc = 0,5
)

Parameters

cwt  IROMatrixDouble
Continuous wavelet transform from which the ridge_lines were defined.
ridge_lines  ListRidgeLine
The ridge lines. Each line consists of a list of tuples (row index, column index, coefficient).
window_sizen  NullableInt32  (Optional)
Size of the window used to calculate the noise floor. Default is cwt.ColumnCount / 20.
min_lengthn  NullableInt32  (Optional)
Minimum length a ridge line needs to be acceptable. Default is cwt.RowCount / 4.
min_snr  Double  (Optional)
Minimum SNR ratio. Default is 1. The signal is the value of the CWT matrix at the shortest length scale (cwt[0, loc]), the noise is the noise_perc percentile of the absolute CWT coefficients in a window around that position.
noise_perc  Double  (Optional)
When calculating the noise floor, percentile of data points (0..1) below which to consider noise. Default value is 0.5.

Return Value

ListRidgeLine
Only those ridge lines that match the criteria.
