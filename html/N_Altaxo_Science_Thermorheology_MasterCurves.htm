ShiftGroupBase
The base of a collection of multiple x-y curves (see ShiftCurveT) that will finally form one master curve.
ShiftGroupCollection
A set of IShiftGroups. The curves in these collections will be shifted with a common set of shift factors. For example, for complex data to be shifted, there will be a ShiftGroupComplexCommonX.
ShiftGroupComplexCommonX
A collection of multiple x-y curves (see ShiftCurveT) that will finally form one master curve. Here, the real and imaginary part of the complex value have a common x-value.
