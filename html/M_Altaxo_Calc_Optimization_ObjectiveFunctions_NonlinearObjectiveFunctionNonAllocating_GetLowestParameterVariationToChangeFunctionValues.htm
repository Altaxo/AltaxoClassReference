NonlinearObjectiveFunctionNonAllocatingGetLowestParameterVariationToChangeFunctionValues Method

If a parameter is zero, it is hard to find the right order of magnitude for a variation of that parameter. Here, the variation is guessed by starting with the lowest possible variation, and increase the variation, until the function values deviate from the original value.
