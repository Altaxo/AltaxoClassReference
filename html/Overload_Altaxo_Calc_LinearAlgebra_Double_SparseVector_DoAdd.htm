DoAdd(Double, VectorDouble) Adds a scalar to each element of the vector and stores the result in the result vector. Warning, the new 'sparse vector' with a non-zero scalar added to it will be a 100% filled sparse vector and very inefficient. Would be better to work with a dense vector instead.
DoAdd(VectorDouble, VectorDouble) Adds another vector to this vector and stores the result into the result vector.
