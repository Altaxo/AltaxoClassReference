Import of existing Markdown files

Previous section: Concatenate two text documents

It is possible to import markdown files. Choose File → Import → Markdown file.. and select the markdown file you want to import. By pressing OK, the Markdown file is imported into a new Text item. Additionally, all images the markdown file refers to are imported as local files into the Text item.
