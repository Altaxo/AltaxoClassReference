Remarks
The user must select an even number of columns. All columns of the first half of the selection must have the same number of rows, and all columns of the second half of selection must also have the same number of rows. The first half of selected columns form a matrix of dimensions(firstrowcount,halfselected), and the second half of selected columns form a matrix of dimension(halfselected, secondrowcount). The resulting matrix has dimensions (firstrowcount,secondrowcount) and is stored in a separate worksheet.
