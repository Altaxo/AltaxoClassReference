Stores a closed polygon and the normals. In order to distinguish between soft vertices and sharp vertices, the sharp vertices are stored twice, because every sharp vertex has two normals. Thus there is a 1:1 relationship between the indices of the normals and the points.

Matrix4x3
Transformation matrix for affine transformations in 3D space. The elements M14, M24 and M34 are assumed to be 0, and M44 is assumed to be 1. Remember: when going from root to leaf node, prepend the new transformation to the existing transformation. And when going from leaf to root node, append the new transformation to the existing one.
Matrix4x4
Transformation matrix for general non-affine transformations in 3D space. This matrix is used mainly for intermediate results.
