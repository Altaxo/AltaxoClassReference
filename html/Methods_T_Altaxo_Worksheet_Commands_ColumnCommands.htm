ExtractPropertiesFromColumn
This function searches for patterns like aaa=bbb in the items of the text column. If it finds such a item, it creates a column named aaa and stores the value bbb at the same position in it as in the text column.
ExtractPropertiesFromString
This function searches for patterns like aaa=bbb in the provided string. If it finds such a item, it creates a column named aaa and stores the value bbb at the same position in it as in the text column.
