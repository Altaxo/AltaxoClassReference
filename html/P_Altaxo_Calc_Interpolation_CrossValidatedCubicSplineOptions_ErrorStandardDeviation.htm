ErrorStandardDeviation Property

CrossValidatedCubicSplineOptionsErrorStandardDeviation Property

If the error standard deviation of the provided points is unknown, set this value to -1 (default value). Then a cross validating cubic spline is fitted to the data. If the error standard deviation is known and is equal for all points, set this value to the error standard deviation of the points (must be greater than zero). If the error standard deviation, provide the error standard deviation for each point (in this case, ErrorStandardDeviation is ignored and internally set to 1). by calling Interpolate(IReadOnlyListDouble, IReadOnlyListDouble, IReadOnlyListDouble)
