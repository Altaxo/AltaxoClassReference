CreateInstanceFromList
Tries to get a class instance for a given attribute type. All loaded assemblies are searched for classes that attributeType applies to, then for all found classes the instantiation of a class is tried, until a instance is created successfully. Here, the attributeType has to implement IClassForClassAttribute, and creationArg[0] has to match the type in TargetType

GetAttributeInstancesAndClassTypesForClass(Type, Object, ReflectionServiceIAttributeForClassListCollection)
For a given type of attribute, attributeType, this function returns the attribute instances and the class types this attributes apply to. If the attribute implements the IComparable interface, the list is sorted. The attribute has to implement the IClassForClassAttribute interface, and only those attributes are considered, for which the TargetType match the type of the target argument.
