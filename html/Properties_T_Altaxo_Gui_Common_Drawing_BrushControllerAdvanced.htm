UseDocumentCopy Sets whether or not a copy of the document is used. If set to true, a copy of the document is used, so if the controller is not applied, all changes can be reverted. If set to false, no copy is made. The document is directly changed by the controller, and changes can not be reverted. Use the last option if a controller up in the hierarchy has already made a copy of the document.
