SolidStraightLineDashSegment Structure

Contains code to generate triangle geometry for solid straight dash segments. This structure needs to be initialized only once per line with Initialize(PenX3D, VectorD3D, VectorD3D, LineD3D). It then can be used for each individual dash segment by calling AddGeometry(ActionPointD3D, VectorD3D, ActionInt32, Int32, Int32, Boolean, Int32, LineD3D, ILineCap, ILineCap).
