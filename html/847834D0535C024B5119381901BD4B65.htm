Concatenate two text documents

Previous section: Usage (of the editor and preview window)

To concatenate two text documents, you can always copy-paste markdown text from one text document to another. But in this way you can not copy local images! To copy both the markdown text and the local images stored into the text document, use the main menu command Edit → Copy Specific → Copy text with images. Then put your cursor at some place in the other text document, and use CTRL-C or Edit → Paste to paste the markdown text and the local images into this text document.
