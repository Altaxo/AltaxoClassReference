GUIFactoryServiceShowDialog Method (Object, String, Boolean)

Shows a configuration dialog for an object.

Namespace:  Altaxo.Main.Services
Assembly:  AltaxoBase (in AltaxoBase.dll) Version: 4.8.2867.0 (4.8.2867.0)
Syntax

public bool ShowDialog(
	Object[] args,
	string title,
	bool showApplyButton
)

Parameters

args
Type: SystemObject
Hierarchy of objects. Args[0] contain the object for which the configuration dialog is searched. args[1].. can contain the parents of this object (in most cases this is not necessary. If the return value is true, args[0] contains the configured object.
title
Type: SystemString
The title of the dialog to show.
showApplyButton
Type: SystemBoolean
If true, the apply button is shown also.

Return Value

Type: Boolean
True if the object was successfully configured, false otherwise.

Implements

IGuiFactoryServiceShowDialog(Object, String, Boolean)
Remarks
The presumtions to get this function working are: A controller which implements IMVCAController has to exist.A UserControllerForObjectAttribute has to be assigned to that controller, and the argument has to be the type of the object you want to configure.A GUI control (Windows Forms: UserControl) must exist, to which an UserControlForControllerAttribute is assigned to, and the argument of that attribute has to be the type of the controller.
