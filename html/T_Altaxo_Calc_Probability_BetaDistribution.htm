Remarks
Copy
Return Beta distributed random deviates according to the density

                a-1       b-1
               x     (1-x)
 p   (x) dx = --------------- dx   for 0 < x < 1
  a,b              B(a,b)

            =  0                   otherwise

References:

R. C. H. Cheng, Generating Beta Variatew with Non-integral Shape
Parameters, Comm. ACM, 21, 317-322 (1978). (Algorithms BB and BC).
