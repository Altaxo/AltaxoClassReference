Type: Altaxo.Calc.LinearAlgebraIMatrixDouble
At the end of this call, contains the Jacobian matrix. The implementer must check if a matrix is provided and must check the dimensions of the provided matrix. In case the provided matrix is null or does not fit the requirements, the implementer must allocate an appropriate matrix. At the end of the call, the matrix must be filled with the elements of the Jacobian.
