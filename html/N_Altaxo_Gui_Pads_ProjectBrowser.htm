This helps in conditions where the number of selected data columns and the type of the item cares. Two attributes are used: selcount and itemtype. The value of selcount is one of the comparer chars: (< = >) and an integer number, for instance: =1. The value of itemtype, if set, is the class name of the items, that should be selected.
