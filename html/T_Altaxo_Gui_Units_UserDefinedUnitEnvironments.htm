UserDefinedUnitEnvironments Class

Collection of unit environments defined by the user. Key is a quantity that is not neccessarily a unit quantity (for instance: unit quanity is 'Length' but quantities can be 'CapSize', 'LineThickness' etc). Value is the user defined unit environment for that quantity.
