A very general flood fill algorithm (4-Neighbour algorithm). It tests iteratively the 4 neighbouring pixels.

IsPixelToBeFilled
Type: SystemFuncInt32, Int32, Boolean
Function that evaluated if the pixel fulfills a condition so that it should be set to a new value. Args are the x and y pixel coordinates, the return value is true if the pixel should be set to a new value.
SetPixelToNewValue
Type: SystemActionInt32, Int32
Action to set the pixel to a new value. Args are the x and y coordinates of the pixel. Since the pixel's value is never tested again, there is no need to really set the pixel's value. Thus, this action can also be used to count the pixels that fulfill the condition given in IsPixelToBeFilled, etc.
