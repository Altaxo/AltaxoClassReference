MVCANControllerEditCopyOfDocBaseTModel, TView Class

Base class of controllers that edit a copy of the document. This means that the document is not connected to the document tree when edited. If you need the document connected to the document tree during editing, consider using MVCANControllerEditOriginalDocBaseTModel, TView instead.
