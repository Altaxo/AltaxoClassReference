The ShiftGroupComplexCommonX type exposes the following members.

Properties
  NameDescription
Count
Gets the number of curves in this shift group.
CreateInterpolationFunction
Creates the fit function. Argument is the tuple consisting of X, Y, and optional YErr. Return value is a function that calculates y for a given x.
FittingWeightImaginary
Gets the fitting weight, a number number > 0.
FittingWeightReal
Gets the fitting weight, a number number > 0.
LogarithmizeXForInterpolation
Logarithmize x values before adding to the interpolation curve. (Only for interpolation).
(Inherited from ShiftGroupBase.)
LogarithmizeYForInterpolation
Logarithmize y values before adding to the interpolation curve. (Only for interpolation).
(Inherited from ShiftGroupBase.)
NumberOfValueComponents
Gets the number of value components. For a usual curve, the return value is 1. For a curve consisting of complex values, the return value is 2.
ParticipateInFitByFitWeight
Gets a value indicating whether this group will participate in the fit, determined by the fit weight. When the fit weigth is zero, the return value is false. Otherwise, if the fit weight is positive, the group principally will participate in the fit, and the return value is true.
XShiftBy
Determines how to shift the x values: either by factor or by offset. Use offset if the original data are already logarithmized.
(Inherited from ShiftGroupBase.)
