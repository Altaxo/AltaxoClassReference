ILineCapContour The contour for a line cap. A list of contour points describes the shape of the cap. The x component of each contour point corresponds to the advance of the cap in line direction, whereas the y-component of each contour point is multiplied with the cross section of the line (i.e. it is the relative thickness of the cap at this point).
