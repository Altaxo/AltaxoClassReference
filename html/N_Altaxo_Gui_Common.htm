SimpleImmutableObjectControllerTObject
Controller for simple immutable objects which have simple properties (double, string) and setter methods following the naming scheme WithPropertyName(propertyName). It is essential that the parameter name of the setter method is the same (but the first letter can be lower case) as the property name. You must subclass this controller and set a attribute at which classes can be controlled.
