InvokeIfRequired(Action)
Invokes the specified callback on the message loop and waits for its completion. If the current thread is the thread running the message loop, executes the callback directly without pumping the message loop.
InvokeIfRequiredT(FuncT)
Invokes the specified callback, waits for its completion, and returns the result. If the current thread is the thread running the message loop, executes the callback directly without pumping the message loop.
